# TruthLens-UA — Demo Test 1 MVP (20:36 10.03.2026)

**Мета:** швидко запустити та протестувати оновлене MVP (API + Streamlit-дешборд + URL-аналіз) на локальному ПК або на іншій Windows-машині під час демо/презентації.

---

## 1. Швидкий локальний запуск (ваш поточний ПК)

### 1.1. Відкрити PowerShell та перейти в репозиторій

```powershell
cd C:\Users\home2\PycharmProjects\PythonProject9
```

### 1.2. Запустити повний стек (API + дашборд, локально)

```powershell
powershell -ExecutionPolicy Bypass -File scripts\run_mvp_demo.ps1 -Mode local
```

**Результат:**
- FastAPI API: `http://127.0.0.1:8000` (`http://localhost:8000/docs` для Swagger).
- Streamlit-дешборд: `http://localhost:8501`.

---

## 2. Запуск демо на іншому Windows-пк (Render як бекенд)

### 2.1. Клонувати репозиторій TruthLens-UA

```powershell
git clone https://github.com/102012dl/TruthLens-UA.git
cd TruthLens-UA
```

### 2.2. Запустити демо з Render API

```powershell
powershell -ExecutionPolicy Bypass -File scripts\run_mvp_demo.ps1 -Mode render
```

**Результат:**
- API: `https://truthlens-ua.onrender.com`
- Swagger: `https://truthlens-ua.onrender.com/docs`
- Streamlit-дешборд: `http://localhost:8501`

## 4. Кроки для демонстрації MVP

1. **Запуск скрипта** (локально або з Render) як описано вище.
2. Відкрити у браузері:
   - `http://localhost:8501` — Streamlit-дешборд.
   - `http://localhost:8000/docs` або `https://truthlens-ua.onrender.com/docs` — Swagger.
3. На вкладці `Analyze` ввести:
   - текст новини / посту **або**
   - URL (наприклад, новина з `tsn.ua`) й натиснути **Analyze URL**.
4. Показати:
   - вердикт (REAL / FAKE / SUSPICIOUS),
   - multi-agent breakdown (Bias / Facts / Credibility),
   - ланцюг аналізу,
   - бізнес-інсайти та рекомендації.
5. Показати Telegram-бота: `https://t.me/truthlens_ai_bot` (через QR з дашборду або посилання).
