## 3. Вартість та ліміти (Cursor AI 20 USD)

- **Підписка Cursor:** фіксована ціна (наприклад $20/міс). Використання Agent (запити, запуск коду, читання файлів) йде в межах підписки.
- **Додаткові платежі:** для роботи з цим проєктом (код, ноутбуки, документація, деплой-інструкції) **не потрібні** — все у межах можливостей Cursor.
- **Рекомендація:** не вводити платіжні дані для додаткових сервісів, якщо це не потрібно; Render/Railway мають безкоштовні плани, Kaggle — безкоштовний доступ до датасетів, GitHub/GitLab — безкоштовні публічні репо.

Конкретні ліміти (скільки запитів/місяць тощо) дивіться в обліковому записі Cursor; тут оцінка лише те, що типова підготовка Capstone **не вимагає додаткових витрат понад підписку**.

## 4. Висновки та рекомендації

1. **Ноутбуки:** У проєкті є `notebooks/01_isot_fake_news_mlflow.ipynb` та `notebooks/03_ua_nlp_training.ipynb`. Для повноцінного A/B тренування: спочатку завантажити ISOT у `data/`, виконати 01 (отримати `artifacts/best_model.joblib`), потім 03 для порівняння моделей та логування в MLflow.
2. **Датасети:** ISOT — через Kaggle (Fake and Real News Dataset); українські дані — HuggingFace `ukr-detect/ukr-twitter-2021` або ручний набір (вже закладено в 03). Скрипт `scripts/download_datasets.py` виводить інструкції.
3. **Git:** Автореєстрації немає. Використайте посилання та кроки з `docs/GIT_SETUP.md` (GitHub + GitLab для 102012dl, email 102012dl@gmail.com).
4. **Render / Railway:** Опис та покроковий деплой — у `docs/DEPLOYMENT.md`. Після push коду достатньо підключити репо та вказати команду старту (наприклад через Procfile).
5. **Захист:** Готові 8 демо-текстів та сценарій 5-хвилинного демо (мастер-промпт v4.0). Рекомендується один раз прогнати їх через API (локально або на Render/Railway) перед захистом.
