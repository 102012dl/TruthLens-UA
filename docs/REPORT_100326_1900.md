## 1. Що оновлено в цьому циклі

### 1.1 Дашборд Streamlit (оновлений Analyze)

- Додано окремі режими запуску аналізу:
  - **Analyze text** — класичний аналіз вставленого тексту.
  - **Analyze URL** — аналіз вмісту сторінки за URL.
- Для URL:
  - Дашборд завантажує HTML через `requests.get(...)`,
  - видаляє HTML-теги (простим `re.sub("<[^>]+>", " ", html)`),
  - нормалізує пробіли та обрізає текст (до ~8000 символів),
  - передає отриманий текст у існуючий ендпоінт `POST /api/analyze`.
- Додано детальний вивід результатів у стилі multi-agent:
  - **Bias / Facts / Credibility** із прогрес-барами.
  - Кроковий **ланцюг аналізу** (preprocessing → IPSO → ML → постпроцесинг → фінальний вердикт).
  - Блок **бізнес-інсайтів та рекомендацій** (для загальних та медичних/health-тем).

Файл: `scripts/truthlens_dashboard.py`.

### 1.2 Гнучкий вибір API (локальний / Render)

- Тепер `truthlens_dashboard.py` читає базовий URL API з змінної середовища:
  - `TRUTHLENS_API_URL` (fallback: `https://truthlens-ua.onrender.com`).
- Це дозволяє:
  - локально використовувати `http://127.0.0.1:8000` без зміни коду;
  - на інших машинах швидко перемикатись між локальним API та Render.

## 2. Єдина команда для запуску MVP після перезапуску ПК

Створено PowerShell-скрипт **`scripts/run_mvp_demo.ps1`**, який:

1. Переходить у корінь репозиторію.
2. Створює `.venv`, якщо його ще немає.
3. Активує `.venv` і встановлює залежності з `requirements.txt`.
4. Залежно від режиму:
   - `Mode = "local"`:
     - стартує локальний API: `uvicorn src.api.main:app --reload` на `http://127.0.0.1:8000`;
     - виставляє `TRUTHLENS_API_URL=http://127.0.0.1:8000` для дашборду.
   - `Mode = "render"`:
     - виставляє `TRUTHLENS_API_URL=https://truthlens-ua.onrender.com`.
5. Стартує Streamlit-дешборд: `streamlit run scripts/truthlens_dashboard.py`.

### Як запускати (Windows / PowerShell)

```powershell
# Локальний режим (рекомендовано для захисту з цього ПК)
powershell -ExecutionPolicy Bypass -File scripts\run_mvp_demo.ps1 -Mode local

# Режим Render (використати задеплоєний API)
powershell -ExecutionPolicy Bypass -File scripts\run_mvp_demo.ps1 -Mode render
```

Після цього:
- API: `http://127.0.0.1:8000` (локально) або `https://truthlens-ua.onrender.com` (Render).
- Дашборд: `http://localhost:8501`.

## 3. Зовнішнє рішення для демонстрації на інших ПК

Для демонстрації на сторонніх ПК (ментор, інші користувачі) достатньо:

1. **Клонувати репозиторій TruthLens-UA** (GitHub або GitLab):
   ```bash
   git clone https://github.com/102012dl/TruthLens-UA.git
   cd TruthLens-UA
   ```
2. **Запустити PowerShell-скрипт**:
   ```powershell
   powershell -ExecutionPolicy Bypass -File scripts\run_mvp_demo.ps1 -Mode render
   ```
   або `-Mode local`, якщо є Python 3.11+ і бажання підняти локальний API.
3. Відкрити в браузері:
   - `http://localhost:8501` — Streamlit-дешборд (Analyze, History, Statistics, QR, URL-аналіз).
   - `https://truthlens-ua.onrender.com/docs` — Swagger API (якщо використовуємо Render).

Telegram-бот `https://t.me/truthlens_ai_bot` продовжує працювати як мобільна точка входу; дашборд надає QR-коди та посилання як раніше.
